## 📌 Overview

A simple and interactive **Recipe Search Application** built using React.js.  
Users can search for recipes by entering ingredients or dish names and get detailed results including ingredients, calories, and images.

---

## 🚀 Features

- 🔍 Search recipes by keyword  
- 🍛 View ingredients list  
- 🖼️ Display recipe images   
- 🌐 Real-time API fetching
